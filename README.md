# REALM

A browser-based, mobile-friendly 2D/text MMORPG inspired by classic EverQuest (Project 1999 era) and Risk-style territory control.

## Features

- **Grid-based world** with zones, cities, and dungeons
- **Class system** with skills and progression (levels 1-10)
- **Combat system** with auto-attack, skills, XP, and loot
- **Faction system** with reputation and guard responses
- **Cloud-saved accounts** with multiple characters per account
- **P99 data import** - Import Project 1999-inspired content

## Quick Start

1. Open `index.html` in a browser
2. Register/login to create an account
3. Create a character
4. Begin your adventure!
